# 区块链作业1代码文档

1. 代码涉及到hashlib库和time库，其中hashlib库用于检验本人算法的正确性，time库用于在挖矿模拟函数中计算挖矿时长。

2. 运行SHA256.py即可。

3. 除算法函数外，代码文件中还有另外两个函数：
   * main(): 输入加密字符串分别使用个人算法和库算法进行哈希，并判断结果是否相同。
   * find(N):输入的参数N表示需要寻找前N位是0的哈希值，找到之后会输出寻找的时间和寻找过的总字符串数量。
